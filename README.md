# k8s-dev-101

AI/SW 개발자를 위한 Kubernetes 개발환경 101 실습 저장소입니다.

## Session 1

- Kind 로컬 Kubernetes 클러스터 생성
- kubeconfig와 context 확인
- kubectl 기본 명령 확인
- Bash 스크립트 작성
- Makefile 작성
- GitHub 협업 흐름 준비

## 자주 쓰는 명령

- 환경 확인: `make check`
- 클러스터 생성: `make cluster-create`
- 상태 확인: `make status`
- 클러스터 삭제: `make cluster-delete`
